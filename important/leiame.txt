Testado rodando em Terminal Ubuntu (WSL2).

PARA RODAR: 
1. Tenha instalado o MongoDB (versão usada: 8.2.2);
2. Crie uma nova conexão de URL "mongodb://localhost:27017/" e nome "academia_bd";
3. Crie um novo banco de dados chamado "academia_bd" e de collection name "academia_bd";
4. Execute o comando: 
   ```java -cp ".;lib/*" AcademiaApp``` // se os arquivos .class já foram compilados. Senão, referencie o arquivo comandos_uteis.txt
